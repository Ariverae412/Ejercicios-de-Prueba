# Instrucciones para ejecutar las pruebas de API

Este proyecto contiene pruebas de API para la autenticación en la página Demoblaze utilizando Karate.

## Requisitos previos

1. **Java JDK 8 o superior**

2. **Apache Maven**

3. **Visual Studio Code**

## Paso a paso para ejecutar el proyecto

1. **Clonar el repositorio**
Clona el repositorio del proyecto en tu máquina local.

2. **Navegar al directorio del proyecto**

3. **Compilar el proyecto**

4. **Ejecutar las pruebas**
En 'src\test\java\com\demoblaze\api\' se encuentra la clase 'KarateRunner.java', que te servirá para realizar los casos de prueba.

5. **Generar reporte HTML**
Las pruebas generan un reporte HTML automáticamente después de la ejecución. El archivo se ubicará en la siguiente ruta: 'target\karate-reports\com.demoblaze.api.AuthTests.html'

6. **Ver el reporte**
Abre el archivo HTML generado en tu navegador preferido para visualizar los resultados.

## Notas adicionales

- Si ocurre algún error durante la ejecución, revisa la consola para obtener más detalles.
- Asegúrate de que todos los endpoints de la API estén disponibles y funcionando.
